

 PHP Editor Main Features:

  - Dynamic linting/analysis while typing (missing semicolons, undefined variables, missing backets, etc)
  - Dynamic code completion -> intelligent suggestions of scope variables
  - Static code completion -> built-in php functions and variables

  - Code quality analysis / Refactoring suggestions -> overcomplicated expressions, suboptimal code, unused symbols

 VS Code Extension

  - PHP test runner (ncrunch)
  - Features listed above
  - Search by symbol (classname, etc to find if not in file with same name)
  - Find usages of variable / class
  - Highlights matching symbols under cursor
  - Project-wide refactoring (rename class, property, etc)
  - Boilerplate code generation
  - Code formatting (on typing ";" or "}")
